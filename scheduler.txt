fields:

queue: APScheduler
tasks: dictionary of dictionary (tree)

functions:

__init__(tasks, ap_dict, interval_dict): Sets the tasks, call init_queue.

ap_dict: dictionary of kw arguments for APSchelduler initialization
interval_dict: dictionary of interval according to TaskType

add_task(name, interval=[int|None]): Add a task into queue. If interval equals zero then run once!

remove_task(name): remove the specified task from queue

init_queue(ap_dict, interval_dict): initalizes the queue using the parameters from configure.py, names are the database path, sets the tasks given as param. 
clear_queue(): Remove all tasks from queue
